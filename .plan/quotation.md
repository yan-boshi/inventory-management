# 核心功能新增报价单页面
## 功能
    - 新增报价单
    - 查询报价单（可根据客户名称、客户代码、报价单号、报价单生成日期查询，默认按日期排序）
    - 更新报价单
    - 删除报价单

## 报价单表属性
    - 报价编号（XSD + -Q + 日期 + 递增数字，例如00001、00002，作为主key）
    - 客户名称
    - 客户代码
    - 报价内容，由编号、产品名称、规格型号、规格描述、单位、数量、单价、合计、状态、备注这些项组成的对象数组转换为json字符串后存储到数据库
    - 报价单生成日期
    - 报价单状态（1：报价中，2：已全部销售，3：已部分销售，4：已取消）
    - 备注
## 新增报价单和更新报价单功能详解
  - 点击新增报价单之后，弹出新增报价单弹窗，弹窗内容如下：
  ### 页面布局
  #### 报价单头部
   - 第一行：深圳市旭思达光电科技有限公司（加大加粗，左右居中）
   - 第二行：报价单（加大加粗，左右居中）
  #### 报价单内容
   - 第一行：报价编号（默认生成）
   - 第二行：客户名称（隐形输入框）
   - 第三行：报价内容表，由编号、产品名称、规格型号、规格描述、单位、数量、单价、合计（自动计算：单价 * 数量 = 合计）、状态（1：报价中，2：已销售）、备注、操作（删除、销售）作为列，使用antDesignVue的table组件，默认一行，可以新增或者删除行，最后一行为含税总价，分别填写大小和数字，表格内容可编辑，可调换顺序，含税总价由每一行的合计相加自动计算
  #### 报价说明
  - 第一行：报价有效期 （隐形文本框，默认为自报价之日起10个工作日）
  - 第二行：送货方式  （下拉选择框，默认为“送货上门”）
  - 第三行：报价单税率（隐形百分比输入框，默认为13%）
  #### 报价单底部
  除保存和取消按钮外，还有添加打印按钮和保存并打印按钮
