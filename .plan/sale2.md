# 销售订单核心功能重新开发
## 功能
    - 新增销售订单
    - 查询销售订单（可根据客户名称、客户代码、报价单号、报价单生成日期查询，默认按日期排序）
    - 更新销售订单
    - 删除销售订单

## 销售订单表属性
    - 销售订单编号（XSD + -S + 日期 + 递增数字，例如00001、00002，作为主key）
    - 客户名称
    - 客户代码
    - 销售商品内容，由编号、业务分类、产品名称、规格型号、规格描述、单位、数量、税率、含税单价、未税单价、含税金额、未税金额、税额、状态、备注这些项组成的对象数组转换为json字符串后存储到数据库
    - 销售日期
    - 销售订单状态（1：正常，2：已退货）
    - 备注
## 新增销售订单和更新销售订单功能详解
  - 点击新增销售订单之后，弹出新增销售订单弹窗，弹窗内容如下：
  ### 页面布局
  #### 销售订单头部
   - 第一行：默认单据编号和销售合同编号
   - 第二行：客户名称和结算方式
  #### 销售订单内容
销售订单内容表，由编号、业务分类、产品名称、规格型号、规格描述、单位、数量、税率、含税单价、未税单价、含税金额、未税金额、税额、状态、备注、操作作为列，使用antDesignVue的table组件，默认一行，可以新增行和删除行，最后一行为含税总价，分别填写大小和数字，表格内容可编辑，可调换顺序，含税总价由每一行的合计相加自动计算
  #### 其他内容
    - 第一行：币种和汇率
    -  第二行：发货日期
    -  第三行：备注
  #### 操作按钮
    - 保存
    - 取消
    - 打印
    - 保存并打印
