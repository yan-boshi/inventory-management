# 采购订单核心功能重新开发
## 功能
    - 新增采购订单
    - 查询采购订单（可根据供应商名称、供应商代码、报价单号、报价单生成日期查询，默认按日期排序）
    - 更新采购订单
    - 删除采购订单

## 采购订单表属性
    - 采购订单编号（XSD + -P + 日期 + 5位随机数字，作为主key）
    - 采购合同编号，由用户填写
    - 供应商名称
    - 供应商代码
    - 采购商品内容，由编号、业务分类、产品名称、产品代码、规格型号、规格描述、单位、数量、税率、含税单价、未税单价、含税金额、未税金额、税额、状态、备注、总价这些项组成的对象数组转换为json字符串后存储到数据库
    - 币种
    - 发货日期
    - 到货日期
    - 采购订单状态（1：采购中，2：已到货）
    - 备注
## 新增采购订单和更新采购订单功能详解
  - 点击新增采购订单之后，弹出新增采购订单弹窗，弹窗内容如下：
  ### 页面布局
  #### 采购订单头部
   - 第一行：默认单据编号和采购合同编号
   - 第二行：供应商名称和结算方式
  #### 采购订单内容 
采购订单内容表，由编号、业务分类、产品名称、产品代码、规格型号、规格描述、单位、数量、税率、含税单价、未税单价、含税金额、未税金额、税额、状态、备注、总价这些项作为列，使用antDesignVue的table组件，默认一行，可以新增行和删除行，最后一行为含税总价，表格内容可编辑，可调换顺序，含税总价由每一行的合计相加自动计算
  #### 其他内容
    -  第一行：币种和汇率
    -  第二行：发货日期和到货日期
    -  第三行：备注
  #### 操作按钮
    - 保存
    - 取消
