# 销售订单打印功能
## 开发装箱清单模板界面
### 页面头
    - 第一行：深圳市旭思达光电科技有限公司
    - 第二行：深圳市龙岗区坂田街道南坑社区五和大道（南）42号景丰大厦602，页面头加大加粗，左右居中
### 页面上面部分
    - 包含订单编号、客户名称、客户地址、合同编号、送货日期，客户名称和客户地址一列，合同编号和送货日期一列，各占据50%宽度，订单编 号在第一行，单独占一行，并且只有订单编号的内容没有项目名，靠左显示
### 页面中间部分：
    - 由序号、产品代码、产品名称、规格描述、数量、单位、备注组成的表格
#### 页面底部
    - 包含总计、快递单号、发货人、联系电话，这4项每项占一行，并且项目名和项目内容分别对齐，发货人从用户信息的用户名中获取，联系电话从用户信息的联系方式中获取
## 销售订单列表的操作栏新增打印功能
    - 点击打印按钮，弹出装箱清单模板界面，并且要把当前销售订单的对应内容填写到模板上，客户地址是客户管理里的客户的收货地址，当快递单号、发货人、联系电话未填写时用隐形的输入框代替
