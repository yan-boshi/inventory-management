# 入库单核心功能
## 功能
    - 新增入库单
    - 查询入库单（可根据产品名称、产品代码、入库单号、入库单生成日期查询，默认按日期排序）
    - 更新入库单
    - 删除入库单

## 入库单表属性
    - 入库单编号（XSD + -W- + 日期 + 5位递增数字，例如00001、00002，作为主key）
    - 采购订单编号
    - 入库商品内容，由序号、产品代码、产品名称、规格描述、单位、数量、备注这些项组成的对象数组转换为json字符串后存储到数据库
    - 入库时间（格式为年月日时）
    - 客户名称
    - 客户地址
    - 总计
    - 币种
    - 入库人
    - 联系电话
    - 备注
## 新增入库单和更新入库单功能详解
  - 点击新增入库单之后，弹出新增入库单弹窗，弹窗内容如下：
  ### 页面布局
  #### 入库单头部
   - 第一行：默认单据编号和采购订单编号（采购订单编号从状态为未入库的采购订单列表中选择，只能选择一个，支持搜索,选择好采购订单编号，自动填充客户名称、客户地址、入库商品内容、总计、币种信息）
   - 第二行：客户名称和客户地址
  #### 入库单内容
入库单内容表，由序号、产品代码、产品名称、规格描述、单位、数量、备注这些项作为列，使用antDesignVue的table组件，默认一行，可以新增行和删除行，可调换顺序
  #### 其他内容
    -  第一行：总计
    -  第二行：币种
    -  第二行：入库时间（默认为当前时间，格式为年月日时）
    -  第三行：备注
  #### 操作按钮
    - 保存
    - 取消
    - 打印
    - 保存并打印
# 打印界面
    - 入库单打印界面和装箱清单打印界面一致，直接用装箱清单打印界面即可
