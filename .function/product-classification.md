# 产品分类功能
## 核心功能
 - 新增产品分类（分为一级分类名、二级分类名、三级分类名，增加一个一级分类名后可以增加多个二级分类名，增加一个二级分类名后可以增加多个三级分类名）
 - 删除产品分类：一级产品分类、二级产品分类、三级产品分类都可以删除
 - 编辑产品分类
 - 查询产品分类（点击产品分类后自动查询所有产品分类，一级、二级、三级产品分类增加检索功能）
## 实现方法
### 产品分类表属性
 - 产品分类（是json对象转换后的json字符串，json对象的结构如下所示）
    一级分类名：{
        二级分类名1：{
            三级分类名1：name1
            三级分类名2：name2
        }
        二级分类名2：{
            三级分类名3： name3
            三级分类名4： name4
        }
    }
 - 分类描述
 - 创建人
 - 创建时间
 - 备注
