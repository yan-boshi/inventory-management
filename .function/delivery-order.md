# 出库单核心功能
## 功能
    - 新增出库单
    - 查询出库单（可根据产品名称、产品代码、出库单号、出库单生成日期查询，默认按日期排序）
    - 更新出库单
    - 删除出库单

## 出库单表属性
    - 出库单编号（XSD + -O- + 日期 + 5位递增数字，例如00001、00002，作为主key）
    - 销售订单编号（从状态为未出库的销售订单列表中选择，只能选择一个，支持搜索）
    - 出库商品内容，由序号、产品代码、产品名称、规格描述、单位、数量、备注这些项组成的对象数组转换为json字符串后存储到数据库
    - 出库时间（格式为年月日时）
    - 送货日期（格式为年月日时）
    - 客户名称
    - 客户地址
    - 总计
    - 币种
    - 出库费用登记类型（由快递费、运杂费、报关费、其他这四种费用组成的json数据转变为json字符串，格式为{"expressDeliveryFee": 100, "transportationFee": 200, "customsFee": 300, "otherFee": 400}）
    - 发货人（默认为当前登录用户）
    - 联系电话（默认为当前登录用户的联系电话）
    - 备注
## 新增出库单和更新出库单功能详解
  - 点击新增出库单之后，弹出新增出库单弹窗，弹窗内容如下：
  ### 页面布局
  #### 出库单头部
   - 第一行：默认单据编号和销售订单编号（销售订单编号从状态为未出库的销售订单列表中选择，只能选择一个，支持搜索,选择好销售订单编号，自动填充客户名称、客户地址、出库商品内容、总计、币种信息）
   - 第二行：客户名称和客户地址
  #### 出库单内容
出库单内容表，由序号、产品代码、产品名称、规格描述、单位、数量、合计金额、备注这些项作为列，使用antDesignVue的table组件，默认一行，可以新增行和删除行，可调换顺序
  #### 出库单底部
    -  第一行：总计
    -  第二行：币种
    -  第二行：出库时间（默认为当前时间，格式为年月日时）
    -  第三行：出库单费用登记
    -  第三行：备注
  #### 操作按钮
    - 保存
    - 取消
    - 打印
    - 保存并打印
# 打印界面
    - 出库单打印界面和销售订单打印界面一致，直接用销售订单打印界面即可
