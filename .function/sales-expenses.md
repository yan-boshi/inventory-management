# 销售管理新增销售费用登记属性
## 销售费用登记分为四种，分别是：
1. 交通费
2. 招待费
3. 礼品费
4. 其他
## 式样 
  打开添加销售订单界面时，销售费用登记位于发货日期下面，四种费用水平排列，查询销售订单界面时，销售费用登记在备注前

## 更新init.sql, 销售订单表新增销售费用登记字段，此字段是一个json字符串
    格式：{"交通费": 100, "招待费": 200, "礼品费": 300, "其他": 400}
```sql
ALTER TABLE sales_orders
ADD COLUMN expenses_expense_type VARCHAR(255) COMMENT '销售费用登记类型';
```
