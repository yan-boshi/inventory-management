
# 核心功能

## 1.销售订单

    - 新增销售订单（客户名称、客户代码、结算方式、业务分类、产品名称、规格型号、规格描述、产品代码需要先获取到数据库中的信息然后以下来列表的形式选择）
    - 查询销售订单（根据单据编号、客户名称、客户代码、销售时间等条件查询，也可查询所有销售订单，查询结果分页显示）
    - 更新销售订单（客户名称、客户代码、结算方式、业务分类、产品名称、规格型号、规格描述、产品代码需要先获取到数据库中的信息然后以下来列表的形式选择）
    - 删除销售订单
    - 退货销售订单

## 2.采购订单

    - 新增采购订单（供应商名称、供应商代码、结算方式、业务分类、产品名称、规格型号、规格描述、产品代码需要先获取到数据库中的信息然后以下来列表的形式选择）
    - 查询采购订单，根据单据编号、供应商名称、供应商代码代码、采购时间等条件查询，也可查询所有采购订单，查询结果分页显示
    - 更新采购订单（供应商名称、供应商代码、结算方式、业务分类、产品名称、规格型号、规格描述、产品代码需要先获取到数据库中的信息然后以下来列表的形式选择）
    - 删除采购订单
    - 退货采购订单

## 3.销售客户管理

    - 新增销售客户
    - 查询销售客户
    - 更新销售客户
    - 删除销售客户
  
## 4.供应商管理

    - 新增供应商
    - 查询供应商
    - 更新供应商
    - 删除供应商

## 5.结算方式管理

    - 新增结算方式
    - 查询结算方式
    - 更新结算方式
    - 删除结算方式

## 6.产品名称管理

    - 新增产品名称
    - 查询产品名称
    - 更新产品名称
    - 删除产品名称

## 7.业务分类管理

    - 新增业务分类
    - 查询业务分类
    - 更新业务分类
    - 删除业务分类

## 8.注册登录功能

    - 注册用户
    - 登录用户
    - 注销用户

## 9.权限管理

    - 新增用户（仅管理员可以新增用户并分配用户的权限，用户权限等级包括管理员、高级用户、普通用户）  
    - 查询用户（仅管理员可以查询用户）
    - 更新用户（仅管理员可以更新用户）
    - 删除用户（仅管理员可以删除用户）
    - 各功能的使用权限：
        - 管理员：所有功能都可以使用
        - 高级用户：销售订单、采购订单、客户管理、供应商管理、结算方式管理、产品名称管理、业务分类管理功能都可以使用
        - 普通用户：销售订单、采购订单功能都可以使用

# 对象属性

## 销售订单属性

    - 默认单据编号（XSD + -S + 日期 + 递增数字，例如00001、00002，作为主key）
    - 销售合同编号
    - 客户名称
    - 客户代码
    - 结算方式
    - 业务分类
    - 产品名称
    - 规格型号
    - 规格描述
    - 产品代码
    - 单位
    - 数量
    - 含税单价
    - 税率
    - 未税单价 （根据含税单价、税率计算，公式为：未税单价 = 含税单价 / (1 + 税率)）
    - 含税销售额（含税，根据数量、含税单价、税率计算，公式为：销售额 = 数量 * 含税单价）
    - 未税金额（根据数量、未税单价计算，公式为：未税金额 = 数量 * 未税单价）
    - 税额（根据含税销售额、税率计算，公式为：税额 = 含税销售额 * 税率）
    - 币种
    - 汇率
    - 发货日期
    - 备注
