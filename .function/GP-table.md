# 报表中心下新增毛利表查询功能
## 功能描述
新增毛利表查询功能，用户可以在报表中心查询毛利表数据。具体逻辑如下：
1. 用户在报表中心选择毛利表查询功能。
2. 系统根据毛利表查询条件，从出库单中获取出库数据，核算出库额，再用出库单的销售合同编号字段拿到销售订单的销售费用登记内，核算销售费用，再从数据库的采购订单表中查询关联销售合同编号等于销售合同编号的采购订单数据，如果查询到采购订单数据，就用采购订单数据的采购订单编号查询入库表中的对应数据，再用入库表数据核算成本。如果没有查询到采购订单数据就用产品管理中对应的商品的单价核算成本
3. 毛利表的列包括：
   - 出货日期
   - 是否结算
   - 结算时间
   - 销售合同编号
   - 销售员
   - 公司名称
   - 结算方式
   - 分类
   - 产品名称
   - 产品代码
   - 规格型号
   - 规格描述
   - 单位
   - 出货数量
   - 单价
   - 销售额（含税）
   - 未税单价
   - 未税金额
   - 采购合同编号
   - 采购员
   - 入库日期
   - 入库数量
   - 入库单价（未税）
   - 入库金额
   - 采购费用（运输费、运营费、增值税、手续费、其他）
   - 销售费用（运输费、手续费、其他）
   - 入库费用（关费、运杂费、报关费、其他）
   - 出库费用（快递费、运杂费、报关费、其他）
   - 毛利（出库额-成本-入库费用中的关税）
   - 提成比例
   - 应发提成
   - 备注
