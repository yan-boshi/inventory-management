# 入库单新增入库费用登记属性
## 入库费用登记分为四种，分别是：
1. 快递费
2. 运杂费
3. 报关费
4. 其他
## 式样 
  打开添加入库单界面时，入库费用登记位于入库时间下面，四种费用水平排列，查询入库单详情界面时，入库费用登记在备注前

## 更新init.sql, 入库单表新增入库费用登记字段，此字段是一个json字符串
    格式：{"快递费": 100, "运杂费": 200, "报关费": 300, "其他": 400}
```sql
ALTER TABLE warehousing_orders
ADD COLUMN expenses_expense_type VARCHAR(255) COMMENT '入库费用登记类型'; 
```
